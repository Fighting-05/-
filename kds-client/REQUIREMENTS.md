# Vue 3 登录页面需求文档

## 技术栈

- Vue 3 (`<script setup>` 语法)
- Tailwind CSS 排版

---

## 1. 视觉追随 (Focus Tracking)

### 猫头鹰捂眼睛效果

在登录卡片上方放置一个简易的 SVG 猫头鹰（或类似的卡通角色）。

- **用户名聚焦**：猫头鹰的眼睛跟随鼠标光标做轻微的平滑转动（用原生 CSS 过渡和 Vue 状态绑定实现）。
- **密码聚焦**：猫头鹰用它的小手遮住眼睛（或整个遮起来）。

---

## 2. 动态背景 (Dynamic Background)

### 柔和流光

- 背景使用**深蓝色底色**，不使用纯色。
- 利用 CSS 实现 **3 个半透明柔和光晕**（Glow Orbs）：
  - 深紫
  - 莫兰迪绿
  - 其他柔和色调
- 光晕在背景中**极其缓慢地漂浮移动**（周期 30s ~ 60s）。
- 保证**护眼、不刺眼、不干扰视线**。

---

## 3. 状态反馈 (State Feedback)

### 按钮反馈

登录按钮点击时有**轻微按压缩放**（Scale）。

### 校验反馈（模拟登录过程）

1. 点击登录按钮
2. 按钮变为 **Loading 状态**
3. **1.5 秒后**，如果输入为空，通过 Vue 逻辑触发 **Shake（抖动）动画**反馈给整个登录卡片

---

## 4. 样式细节

登录卡片需要具备**玻璃拟态（Glassmorphism）**质感：

- `backdrop-blur`
- `bg-white/5`
- 半透明边框
- 阴影效果

---

## 5. 补充要求

- 添加注册功能
- 高级的视觉互动效果
- 丰富的动态背景元素（星星粒子、网格线、极光等）
- 猫头鹰需具备多层次的生动细节（渐变填充、眨眼、耳抖、呼吸、偷看等）
- 卡片需具备 3D 透视倾斜效果
